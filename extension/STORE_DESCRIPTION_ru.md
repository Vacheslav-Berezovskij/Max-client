# MAXSEC — приватный режим для MAX Web

## Короткое описание (до ~132 символов)
MAXSEC добавляет локальное шифрование в веб-чат MAX: режимы OFF/READ/SECURE, ручная проверка ключей, без собственного сервера.

## Полное описание
MAXSEC — это браузерное расширение для web.max.ru, которое добавляет поверх обычного чата опциональный приватный режим обмена сообщениями.

### Что умеет
- Режимы работы:
  - **OFF** — без шифрования и расшифровки
  - **READ** — только расшифровка входящих
  - **SECURE** — шифрование исходящих + расшифровка входящих
- Плавающая панель в чате: текущий режим, fingerprint, статус верификации, кнопка Verify
- Ручная crypto-панель для тестов: Encrypt/Decrypt/Copy без зависимости от DOM мессенджера
- Работа через обычные сообщения чата (in-band), без собственного backend

### Важные принципы
- **Шифрование выполняется только на клиенте (в вашем браузере).**
- Расширение **не отправляет ваши ключи на наши серверы** (у нас нет сервера для ключей/сообщений).
- Ключи и настройки хранятся локально в storage браузера.

### Для кого
- Для пользователей, которым нужен дополнительный приватный слой в веб-версии MAX
- Для команд/тестировщиков, которые хотят валидировать сценарии шифрования до production-интеграции

### Ограничения
- Расширение зависит от структуры DOM веб-клиента MAX; при крупных обновлениях интерфейса может потребоваться обновление расширения.
- При ошибках парсинга/шифрования расширение переключается в безопасный fallback и не должно блокировать обычные сообщения.
